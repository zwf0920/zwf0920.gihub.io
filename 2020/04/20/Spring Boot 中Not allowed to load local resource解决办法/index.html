Spring Boot 中Not allowed to load local resource:解决办法

笔者初学spring boot,最近做了一个上传图片的功能，将图片储存在/src/resourse/static/upload文件夹下，然后直接从d盘根目录开始读取，我的图片地址如下：D:/node/luffydemo/src/main/resources/static/uploadnewPIC.jpg，浏览器能直接访问，但是如果嵌入到项目中会报如下错误：

原因大概是不能直接访问项目外的文件，可能是spring本身做了什么限制，作为初学者的我还不是很了解

解决办法：

新建一个配置类来修改spring 的配置，应该算是一个重定向吧，这里的@configuration顾名思义是一个配置类的注解，会在项目启动时加载。

此时将你的图片地址替换为：http://localhost:8282/static/image/uploadnewPIC.jpg 即可
